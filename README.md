# 自动截图粘贴工具 (CopyPast Tool)

这是一个基于 Electron + React + nut.js 开发的自动化工具，用于自动从屏幕右侧区域截图并粘贴到左侧窗口。

## 功能特点
- **自动截图**：自定义截图区域，不压缩画质。
- **模拟操作**：自动模拟鼠标点击、Ctrl+V 粘贴、PageDown 翻页。
- **循环执行**：支持设置循环次数或无限循环。
- **快捷键停止**：全局快捷键 `Alt+Q` 紧急停止。

## 安装与运行

### 1. 安装依赖
```bash
npm install
```
*如果下载慢，请使用淘宝源：`npm config set registry https://registry.npmmirror.com`*

### 2. 开发模式运行
```bash
npm run dev
```

### 3. 打包生成 .exe
```bash
npm run build
```
打包后的文件位于 `dist-electron/` 目录下，直接运行安装包即可。

## 使用说明

1. **校准位置**：
   - 点击 **① 拾取左上角**：倒计时3秒后，记录鼠标当前位置为右侧截图区域的左上角。
   - 点击 **② 拾取右下角**：倒计时3秒后，记录右下角，计算出截图宽高。
   - 点击 **③ 拾取点击位置**：倒计时3秒后，记录左侧窗口的点击粘贴位置。

2. **设置参数**：
   - 设置操作间隔时间（建议 1000ms 以上）。
   - 设置循环次数。

3. **开始任务**：
   - 点击“开始任务”按钮。
   - 程序将自动执行：截图 -> 点击左侧 -> 粘贴 -> 左侧翻页 -> 点击右侧 -> 右侧翻页 -> 循环。

4. **停止任务**：
   - 点击界面上的“停止”按钮。
   - 或按下键盘 **`Alt + Q`** 强制停止。

## 注意事项
- 运行过程中请勿随意移动鼠标，以免干扰自动化操作。
- 确保左右两个软件窗口处于前台可见状态。
